---
title: "Get Day-wise Call Counts"
api: "GET /getDayWiseCallCounts"
description: "Retrieve aggregated call statistics grouped by day for analytics and reporting."
---

## Headers

<ParamField header="x-api-key" type="string" required>
  Your VoiceGenie API key for authentication
</ParamField>

## Query Parameters

<ParamField query="start_date" type="string" required>
  Start date for analytics (ISO 8601 format)
</ParamField>

<ParamField query="end_date" type="string" required>
  End date for analytics (ISO 8601 format)
</ParamField>

<ParamField query="campaign_id" type="string">
  Optional: Filter by specific campaign
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://seamless-sandbox-dev-cm.oriserve.com/getDayWiseCallCounts?start_date=2024-01-01&end_date=2024-01-31' \
  --header 'x-api-key: your-api-key-here'
```

</RequestExample>

<ResponseExample>
