---
title: "Check Campaign Name Existence"
api: "POST /checkCampaignNameExistence"
description: "Check if a campaign name is already taken."
---

## Quick Start

Validate campaign names before creation to prevent duplicate naming conflicts. This is essential for maintaining unique campaign identities.

<CardGroup cols={2}>
  <Card title="Pre-validation" icon="check">
    Check availability before campaign creation
  </Card>
  <Card title="Case-Sensitive" icon="shield">
    Names are case-sensitive for flexibility
  </Card>
</CardGroup>

<Note>
  **Soft-deleted campaigns** also count as taken names in the system.
</Note>

<Info>
  **API Playground:** Use the interactive playground to test campaign name availability instantly.
</Info>

  Origin of the request (e.g., https://vb-seamless-dev-backend.oriserve.com/)

  Browser or client user agent
</ParamField>

## Request Body

<ParamField body="campaignName" type="string" required>
  Campaign name to check
  
  Example: `Q4 Sales Campaign`
</ParamField>

## Response

<ResponseField name="status" type="number">
  HTTP status code (200 if available, 409 if exists)

<ResponseField name="message" type="string">

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://vb-seamless-dev-backend.oriserve.com/checkCampaignNameExistence' \
  --header 'Content-Type: application/json' \
  --cookie 'token=your_auth_token' \
  --data '{
    "campaignName": "Q4 Sales Campaign"
  }'
```

```javascript JavaScript
const campaignName = "Q4 Sales Campaign";

const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  credentials: 'include',
  body: JSON.stringify({ campaignName })
};

fetch('https://vb-seamless-dev-backend.oriserve.com/checkCampaignNameExistence', options)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
```

```python Python
import requests

response = requests.post(
    'https://vb-seamless-dev-backend.oriserve.com/checkCampaignNameExistence',
    headers={'Content-Type': 'application/json'},
    cookies={'token': 'your_auth_token'},
    json={'campaignName': 'Q4 Sales Campaign'}
)
print(response.json())
```


## Notes

- Campaign names are case-sensitive
- Soft-deleted campaigns also count as taken
