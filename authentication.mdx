# Authentication

All API requests to VoiceGenie require authentication using an API key.

## API Key Authentication

Include your API key in the request headers:

```bash
x-api-key: your-api-key-here
```

## How to Get Your API Key

1. Log in to your VoiceGenie dashboard
2. Navigate to Settings > API Keys
3. Generate a new API key or copy your existing key

## Security Best Practices

- **Never share your API key** in public repositories or client-side code
- **Rotate your keys regularly** for enhanced security
- **Use environment variables** to store your API keys
- **Implement IP whitelisting** when possible

## Example Request

```bash
curl --request GET \
  --url https://your-domain.com/api/v1/fetchCampaignData \
  --header 'x-api-key: your-api-key-here'
```

## Error Responses

If authentication fails, you'll receive one of the following errors:

### 401 Unauthorized
```json
{
  "success": false,
  "message": "Invalid or missing API key"
}
```

### 403 Forbidden
```json
{
  "success": false,
  "message": "API key does not have permission to access this resource"
}
```

### 429 Too Many Requests
```json
{
  "success": false,
  "message": "Rate limit exceeded. Please try again later."
}
```
